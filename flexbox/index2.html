Guia Avançado de Flexbox

Principais Propriedades do Flexbox:

  • display: Define um contêiner como um contêiner flexível.
  • flex-direction: Define a direção dos itens no contêiner (row, row-reverse, column, column-reverse).
  • justify-content: Alinhamento dos itens ao longo do eixo principal (flex-start, flex-end, center, space-between, space-around).
  • align-items: Alinhamento dos itens ao longo do eixo transversal (flex-start, flex-end, center, baseline, stretch).
  • flex-wrap: Define se os itens devem quebrar em várias linhas ou não (nowrap, wrap, wrap-reverse).
  • flex: Define a capacidade de um item flexível crescer ou encolher para preencher o espaço disponível.

Instruções de Funcionamento:

  1. Crie um contêiner e defina display: flex nele.
  2. Aplique propriedades como flex-direction, justify-content, e align-items conforme necessário para posicionar os itens.
  3. Estilize os itens individuais usando propriedades como flex para controle do tamanho, order para alterar a ordem, entre outras.
  4. Experimente também as propriedades de alinhamento de itens individuais, como align-self.
